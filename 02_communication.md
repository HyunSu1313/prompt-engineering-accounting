# 02. 커뮤니케이션용 프롬프트

검토멘트, 감사조서 문구, 사내 메일 등 **대내 커뮤니케이션**과
클라이언트 인터뷰 질문, 자료 요청 문장 등 **대외 커뮤니케이션**을
실무에서 바로 쓸 수 있는 형태로 생성·검토하기 위한 프롬프트 템플릿입니다.

> **📌 준용 기준**
> 본 템플릿의 모든 답변은 아래 기준을 우선 준용합니다.
> - 한국채택국제회계기준(K-IFRS) 또는 일반기업회계기준(K-GAAP)
> - 내부회계관리제도 설계·운영 개념체계 (금융감독원)
> - 내부회계관리제도 평가 및 보고 가이드라인 (금감원)

---

## 📌 사용 목적

**대내**
- 검토멘트·감사조서 문구 다듬기
- 사내 메일, 상급자 보고 문장 정리

**대외**
- 클라이언트 인터뷰 질문 작성
- 자료 요청 문장, 확인서 문구 작성

---

## 📎 정확한 원문 인용을 위한 PDF 첨부 안내

회계처리 또는 주석공시 관련 표현이 포함된 경우, 관련 기준서 PDF를 첨부하면
사용 중인 AI가 해당 문단 원문을 정확하게 인용합니다.

| 기준 | 출처 |
|---|---|
| K-IFRS 각 기준서 | kasb.or.kr |
| 내부회계관리제도 설계·운영 개념체계 | k-icfr.org |
| 내부회계관리제도 평가 및 보고 가이드라인 (금감원) | fss.or.kr |

> ⚠️ PDF 미첨부 시 AI가 원문을 추정하여 출력할 수 있습니다.
> 이 경우 반드시 실제 기준서와 대조·검증 후 사용하세요.

---

## 📝 기본 템플릿 — 문장 검토 (대내/대외 공통)

```
다음 문장을 실무 관점에서 검토해줘.

[문장 유형]
{검토멘트 / 감사조서 문구 / 사내 메일 / 클라이언트 인터뷰 질문 / 자료 요청 문장 / 기타}

[원문]
{검토받을 문장}

[출력 형식]
1. 어색한 부분 및 이유
2. 개선 문장 (바로 사용 가능한 형태)
3. 추가 고려사항 (수신자, 격식 수준, 문서 성격에 따른 조언)
   - 회계처리 또는 주석공시 관련 표현이 포함된 경우 관련 근거 첨부:
     기준서명 문단 번호
     원문: "해당 문단 원문 문장"

전문용어는 유지하되, 실무자가 바로 활용할 수 있는 수준으로 작성해줘.
```

---

## ✅ 사용 예시 1 — 검토멘트 (대내)

**입력:**
```
[문장 유형] 검토멘트

[원문]
당사는 확정기여형 퇴직연금제도를 운용하고 있는바, 당기 납부한 기여금을
퇴직급여 비용으로 인식하였으며 이는 적정한 것으로 판단됩니다.

[출력 형식]
1. 어색한 부분 및 이유
2. 개선 문장
3. 추가 고려사항
```

**출력 예시:**

1. **어색한 부분**
   - "운용하고 있는바" → 구어체와 문어체가 혼재. "운영하고 있으며"가 자연스러움
   - "적정한 것으로 판단됩니다" → 수동적 표현. "적정함을 확인함"이 조서 어조에 맞음

2. **개선 문장**
   > "동사는 확정기여형 퇴직연금제도를 운영하고 있으며, 당기 납부 기여금 XXX원을 퇴직급여로 비용 인식함. K-IFRS 제1019호 문단44에 따른 처리로 적정함을 확인함."

3. **추가 고려사항**
   - 감사조서는 "~함", "~확인함" 종결어미 통일 권장
   - 금액 명기 가능 시 구체적 수치 포함

---

## ✅ 사용 예시 2 — 클라이언트 인터뷰 질문 (대외)

**입력:**
```
[문장 유형] 클라이언트 인터뷰 질문

[상황]
유형자산 구매 프로세스 인터뷰를 진행해야 해.
자본적지출과 수익적지출 구분 기준, 승인 권한, 자산 등록 절차를
실무자가 부담 없이 답할 수 있는 질문으로 만들어줘.

[출력 형식]
1. 인터뷰 질문 리스트 (도입 → 본질문 → 확인 질문 순서)
2. 질문 의도 (각 질문이 어떤 통제를 확인하기 위한 것인지)
```

**출력 예시:**

1. **인터뷰 질문 리스트**
   - (도입) "유형자산을 새로 구입하실 때 보통 어떤 절차로 진행되나요?"
   - (본질문) "자본적지출과 수익적지출은 어떤 기준으로 구분하시나요? 예를 들어 금액 기준이 있나요?"
   - (본질문) "구매 요청부터 최종 승인까지 누가 어떤 단계를 거치나요?"
   - (확인) "승인된 이후 자산 등록은 어느 시스템에서, 누가 입력하시나요?"

2. **질문 의도**
   - 도입 질문: 라포 형성 + 전체 프로세스 그림 파악
   - 자본적/수익적지출 질문: 회계처리 정확성 관련 통제 확인
   - 승인 단계 질문: 권한분리 통제 존재 여부 확인
   - 자산 등록 질문: 입력 통제 및 시스템 상 권한 확인

---

## ✅ 사용 예시 3 — 자료 요청 문장 (대외)

**입력:**
```
[문장 유형] 자료 요청 문장

[상황]
재구축 평가를 위해 유무형자산 관련 RCM 초안과 최근 3개월 자산 취득·처분 내역을
클라이언트 실무자에게 요청해야 해. 너무 딱딱하지 않게, 그렇지만 명확하게.

[출력 형식]
1. 요청 메일 또는 메시지 문장
2. 추가 고려사항
```

**출력 예시:**

1. **요청 문장**
   > "안녕하세요, OOO 팀장님. 내부회계관리제도 재구축 평가 관련하여 유무형자산 프로세스 검토를 위해 아래 자료 요청드립니다.
   > 1) 기존 RCM(유무형자산 부분)
   > 2) 최근 3개월간 자산 취득·처분 내역 (전표 또는 리스트)
   > 가능하시면 이번 주 금요일까지 회신 부탁드립니다. 감사합니다."

2. **추가 고려사항**
   - 마감 기한은 구체적 요일·날짜로 명시할 것
   - 자료 형태(엑셀/PDF)를 미리 명시하면 회신 속도 향상

## 🔧 변형 템플릿 — 문장 새로 작성 요청

```
다음 상황에 맞는 문장을 작성해줘.

[문장 유형]
{검토멘트 / 감사조서 문구 / 사내 메일 / 클라이언트 인터뷰 질문 / 자료 요청 문장}

[상황]
{전달할 내용 또는 맥락}

[조건]
- 격식 수준: {높음 / 보통 / 낮음}
- 수신자: {상급자 / 동료 / 외부 클라이언트}
- 분량: {한 줄 / 2~3줄 / 단락}

바로 사용 가능한 형태로 작성해줘.
```
